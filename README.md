# PopX ReactJS UI

Pixel-perfect recreation of the PopX mobile UI screens using React + Vite.

## Screens
- Landing
- Login
- Signup
- Profile

## Tech Stack
- React 18
- React Router
- Vite

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deployment (Vercel)
- Framework preset: Vite
- Build command: npm run build
- Output directory: dist

## Notes
- The UI is centered on desktop and full-bleed on mobile.
- Profile avatar uses a cropped asset at public/avatar.png.
